quality

fun quality(quality: ImageQuality): CdnPathBuilder

Image quality affects size of image and loading speed. Has no effect on non-JPEG images, but does not force format to JPEG.

Parameters

quality - QUALITY_NORMAL – used by default. Fine in most cases. QUALITY_BETTER – can be used on relatively small previews with lots of details. ≈125% file size compared to normal image. QUALITY_BEST – useful if you're a photography god and you want to get perfect quality without paying attention to size. ≈170% file size. QUALITY_LIGHTER – can be used on relatively large images to save traffic without significant quality loss. ≈80% file size. QUALITY_LIGHTEST — useful for retina resolutions, when you don't wory about quality of each pixel. ≈50% file size.
